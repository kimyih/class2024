01. 변수 : 데이터 저장


자바스크립트에서 변수는 값을 저장하고 참조하는 데 사용됩니다.
변수를 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용할 수 있습니다.
변수를 선언하고 값을 할당하여 사용할 수 있습니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 데이터 변경


JavaScript에서 변수는 데이터를 저장하고 변경하는 데 사용됩니다.
변수를 사용하면 프로그램에서 값을 저장하고 나중에 필요할 때 해당 값을 사용할 수 있습니다.
데이터 변경은 변수에 새로운 값을 할당하여 이루어집니다.
이를 통해 변수의 값은 언제든지 변경될 수 있습니다 

{
    let x = 100;            //변수 x에 숫자 100을 저장
    let y = 200;            // 변수 y에 숫자 200을 저장
    let z = "javascript";   //변수 z에 문자 javacripts를 저장

    x = 300;        // 변수 x에 새로운 값 300을 할당
    y = 400;        // 변수 y에 새로운 값 400을 할당
    z = "jquery"    // 변수 z에  "jquery" 문자열을 할당

    console.log(x);     //변수 x의 값인 300을 출력
    console.log(y);     //변수 x의 값인 400을 출력
    console.log(z);     //변수 z의 값인 "jquery"를 출력
}
결과 확인하기
300
400
jquery

변수 x와 y는 새로운 값으로 변경되었고, 변수 z는 "jquery"로 변경

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가


JavaScript 변수는 데이터 저장소로 작용하며, 값을 저장하고 변경할 수 있습니다.
데이터 변경은 변수에 새로운 값을 할당하여 이루어지며,
데이터 추가는 객체나 배열과 같은 자료구조에 새로운 속성이나 요소를 추가하여 이루어집니다. 

{
    let x = 100;        //변수 x에 숫자 100을 저장
    let y = 200;        //변수 y에 숫자 200을 저장
    let z = "javascript";       //변수 z에 문자 "javacript" 을 저장

    x = x + 100;         //변수 x의 값에 100을 더하고 그 결과를 다시 변수 x에 저장
    y = y + 100;         //변수 y의 값에 100을 더하고 그 결과를 다시 변수 y에 저장
    z = z + "jqwery"    //변수 z의 값에 "jquery" 문자열을 더하고 그 결과를 다시 변수 z에 저장

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
200
300
javacriptsjquery

변수 x와 y는 각각 100씩 증가하였고, 변수 z는 "javascript"와 "jquery"를 합친 문자열로 변경

04. 상수 : 데이터 저장 + 데이터 변경(X)


상수는 데이터를 저장하는 데 사용되지만, 한 번 값을 할당하면 변경할 수 없는 변수입니다.
상수는 데이터 저장에 사용되며, 그 값을 변경할 수 없습니다.
JavaScript에서는 const 키워드를 사용하여 상수를 선언합니다. 한 번 할당된 후에는 다시 할당할 수 없으므로 상수의 값은 변하지 않습니다. 

{
    const x = 100;
    const y = 200;
    const z = "javascript";

    console.log(x);
    console.log(y);
    console.log(z);        
}
결과 확인하기
100
200
javascript

상수로 선언된 변수는 초기에 한 번만 값을 할당할 수 있으며, 그 후에는 변경할 수 없다.

05. 배열 : 데이터 저장(여러개) : 표현방법1 : 배열 선언


JavaScript의 배열은 여러 데이터를 하나의 변수에 저장하는데 사용되며, 대괄호를 사용하여 선언됩니다.
예를 들어, let myArray = [1, 2, 3];와 같이 사용됩니다.
이러한 배열은 인덱스를 사용하여 개별 요소에 접근할 수 있습니다. 

{
    const arr = new Array();    // const arr 배열을 new Array 로 선언 

    arr[0] = 100;        //arr 배열에 숫자100을 저장
    arr[1] = 200;        //arr 배열에 숫자200을 저장
    arr[2] = "javascript";       //arr 배열에 문자 javacript를 저장

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
결과 확인하기
100
200
javascript

배열은 인덱스를 사용하여 요소에 접근하고 값을 할당할 수 있다.
이 예제에서는 배열의 각 인덱스에 값을 할당한 후, 해당 인덱스에 있는 값을 출력

06. 배열 : 데이터 저장(여러개) : 표현방법2 : 배열 선언과 동시에 초기값 설정


new Array()를 사용하여 배열을 생성하고, 괄호 안에 초기값을 나열합니다.
그 후에는 생성된 배열을 arr이라는 변수에 할당합니다.
배열의 각 요소는 인덱스를 사용하여 접근할 수 있으며, 인덱스는 0부터 시작합니다.
따라서 arr[0], arr[1], arr[2]는 각각 배열의 첫 번째, 두 번째, 세 번째 요소를 출력합니다. 

{
    const arr = new Array(100, 200, "javascript");  // const arr 배열선언과 동시에 초기값을 입력

    console.log(arr[0]);    // 배열선언 한 초기값에 100출력
    console.log(arr[1]);    
    console.log(arr[2]); 
}
결과 확인하기
100
200
javascript

07. 배열 : 데이터 저장(여러개) : 표현방법3 : 리터럴 방식으로 배열을 선언


리터럴 방식은 값을 코드 안에 직접 표기하는 방법을 가리킵니다.
즉, 변수나 객체를 따로 생성하지 않고 값을 그대로 표기하는 것을 말합니다.
JavaScript에서 배열을 리터럴 방식으로 선언할 때는 대괄호 [ ] 안에 직접 데이터를 나열하여 배열을 생성하는 방식을 의미합니다.
이는 간편하게 배열을 초기화할 수 있는 방법입니다. 

{
    const arr = [];

    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
결과 확인하기
100
200
javascript

08. 배열 : 데이터 저장(여러개) : 표현방법 4 : 리터럴 방식으로 배열 선언과 동시에 초깃값 설정


리터럴 방식으로 배열을 선언하면서 동시에 초기값을 설정할 수 있습니다.
대괄호 [ ] 안에 데이터를 나열하여 배열을 생성하는 방식으로 간편하게 초기화할 수 있습니다.

예를 들어, [1, 2, 3]와 같이 사용하여 배열을 생성하고 초기값을 설정할 수 있습니다. 

{
    const arr = [100, 200, "javascript"];

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
결과 확인하기
100
200
javascript

[100, 200, "javascript"] 형태로 배열을 선언하면, 배열이 생성되고 동시에 초기값으로 요소들이 할당

09. 배열 : 데이터 저장(여러개) : 표현방법 5 : push () 메서드를 이용한 배열 방법


push() 메서드는 JavaScript 배열에 요소를 끝에 추가하는 역할을 합니다.
새로운 데이터를 배열에 넣고 싶을 때 사용하며, 배열의 길이를 자동으로 조정합니다.
이를 통해 동적으로 데이터를 확장하고 추가할 수 있습니다. 

{
    const arr = [100, 200, "javascript"];

    arr.push(400);

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);    // 배열의 세 번째 요소 출력: "javascript"
    console.log(arr[3]);
}
결과 확인하기
100
200
javascript

10. 배열 : 데이터 저장(여러개) : 표현방법 6 : 다차원 배열


다차원 배열은 배열 안에 배열이 들어 있는 것이며, 데이터를 구조화하여 저장하는 데 사용됩니다.
예를 들어, 2차원 배열은 행과 열로 구성된 표를 나타내며, 3차원 배열은 면, 행 및 열로 구성된 데이터를 저장합니다.
각 차원의 배열은 인덱스를 사용하여 데이터에 접근할 수 있습니다. 

{
    const arr = [100, [200, 300,], ["javascript", "jquery"]];

    console.log(arr[0]);   
    console.log(arr[1][0]);     // 배열의 두 번째 요소의 첫 번째 요소 출력: 200 
    console.log(arr[1][1]); 
    console.log(arr[2][0]);     // 배열의 세 번째 요소의 첫 번째 요소 출력: "javascript" 
    console.log(arr[2][1]); 
}
결과 확인하기
100
200
300

11. 배열 : 데이터 저장(여러개) : 표현방법 7 : 배열구조분해할당


배열 구조 분해 할당은 배열의 요소를 분리하여 각각의 변수에 할당하는 방법입니다.
이를 통해 배열의 각 요소에 쉽게 접근하고 변수에 할당할 수 있습니다.
JavaScript에서는 이를 위해 대괄호 []를 사용하여 배열을 선언하고, 할당 연산자인 등호 =와 함께 사용합니다. 

{
    const arr = [100, 200, "javascript"];

    const [a, b, c] = arr;      // 배열 해체를 통해 arr의 요소들을 추출하여 변수 a, b, c에 할당

    console.log(a);
    console.log(b);
    console.log(c);
}
{
     const arr = [100, 200, [300, 400]];
    const [a, b, [x, y]] = arr;         // 배열 해체를 통해 arr의 요소들을 추출하여 변수에 할당

    console.log(a);     // 배열의 첫 번째 요소인 100을 출력
    console.log(b);
    console.log(x);     // 내부 배열의 첫 번째 요소인 300을 출력
    console.log(y);
}

12. 배열 : 데이터 저장(여러개) : 표현방법 8 : 배열펼침연산자


배열 펼침 연산자는 배열 안의 요소들을 풀어서 개별적으로 추출하거나 다른 배열과 결합할 때 사용됩니다.
이 연산자는 대괄호 [] 안에서 ...로 표시됩니다.
주로 배열을 복사하거나 합칠 때 유용하게 사용됩니다. 

{
    const arr1 = [100, 200, "javascript"];
    const arr2 = [300, 400, "jquery"];      
    const arr3 = [...arr1, 500];

    console.log(arr1);      // 배열 arr1을 그대로 출력 [100, 200, "javascript"] 
    console.log(...arr1);       // 배열 arr1을 해체하여 각 요소를 개별적으로 출력 100 200 javascript   
    console.log(...arr1, ...arr2)       // 배열 arr1과 arr2를 해체하여 각 요소를 개별적으로 출력 100 200 javascript 300 400 jquery   
    console.log(...arr3)        //배열 arr3을 해체하여 각 요소를 개별적으로 출력 100 200 javascript 500
}
결과 확인하기
100, 200, javascript
100200javascript
100200javascript 300400jquery
100200javascript500

...의 유무 차이는 결과값 쉼표 유무 차이

13. 객체 : 데이터 저장(키와값) : 표현방법 1 : 객체 선언 후 배열 방식으로 데이터 입력


객체를 선언한 후 배열 방식으로 데이터를 입력하면 숫자 인덱스를 사용하여 값에 접근할 수 있는 JavaScript 객체가 생성됩니다.
이 방법은 주로 순서가 있는 데이터를 객체로 표현할 때 사용됩니다.

객체를 배열 방식으로 데이터를 입력하는 것은 각 데이터에 숫자 인덱스를 부여하여 값을 할당하는 것입니다.
이를 위해 빈 객체를 먼저 생성한 후, 대괄호를 사용하여 각 요소에 값을 할당합니다.
이 방법은 순서가 있는 데이터를 객체에 쉽게 저장할 수 있습니다. 

{
    const obj = new Object();   // 객체 선언

    obj[0] = 100;   //obj에 100 저장
    obj[1] = 200;   //obj에 200 저장
    obj[2] = "javascript";  //obj에 문자 javacript를 저장 

    console.log(obj[0]);
    console.log(obj[1]);
    console.log(obj[2]);    
}

200

14. 객체 : 데이터 저장(키와값) : 표현방법 2 : 객체 선언 후 객체 방식으로 데이터 입력


객체를 선언한 후 객체 방식으로 데이터를 입력하는 것은 키와 값을 직접 지정하여 데이터를 입력하는 것입니다.
중괄호 {} 안에 키와 값을 쉼표로 구분하여 나열하며, 이를 통해 객체를 초기화합니다.
이 방식은 데이터의 의미를 명확하게 표현할 수 있고, 코드의 가독성을 높일 수 있습니다.

{
    const obj = new Object(); // 객체 선언

    obj.b = 200;
    obj.c = "javascript";

    console.log(obj.a);
    console.log(obj.b);
    console.log(obj.c);
}

배열 안에 객체를 저장하는 것은 여러 개의 객체를 하나의 배열에 저장하는 방식입니다. 각 객체는 중괄호 {}로 선언되며, 배열의 요소로 추가됩니다.
이러한 구조를 통해 배열 안에 여러 객체를 쉽게 관리하고 접근할 수 있습니다.

{

    console.log(obj2.c);    //javascript

23. 파일 : 데이터 교환 (JSON)


JavaScript 파일의 데이터 교환(JSON)은 데이터를 JavaScript 객체와 JSON 문자열 간에 변환하고, 웹 애플리케이션과 서버 간에 데이터를 주고받는 데 사용되는 방법입니다.
JSON은 JavaScript Object Notation의 약자로, 데이터를 표현하고 교환하기 위한 경량의 형식입니다.
이는 사람이 읽고 쓰기 쉽고, 기계가 분석하고 생성하기 쉽도록 디자인되어 있습니다.
JavaScript에서는 JSON.stringify()를 사용하여 JavaScript 객체를 JSON 문자열로 변환하고, JSON.parse()를 사용하여 JSON 문자열을 JavaScript 객체로 변환할 수 있습니다.
이를 통해 서로 다른 시스템 간에 데이터를 교환하고 이해하기 쉽도록 할 수 있습니다. 

{
    {
        // 01 JSON 데이터 --> 객체 변환 
        // JSON 형식의 문자열을 JavaScript 객체로 변환합니다.
        const jsonString = '{"name": "Hwang", "age": 30}';
        const jsonObject = JSON.parse(jsonString);
    
        console.log(jsonObject.name); // JSON 객체의 'name' 속성 값을 출력합니다. (결과: "Hwang")
        console.log(jsonObject.age);  // JSON 객체의 'age' 속성 값을 출력합니다. (결과: 30)
    }
    
    {
        // 02 객체 파일 --> JSON 데이터 
        // JavaScript 객체를 JSON 형식의 문자열로 변환합니다.
        const person = { name: "Hwang", age: 30 };
        const jsonString = JSON.stringify(person);
    
        console.log(jsonString); // JavaScript 객체를 JSON 형식으로 변환한 문자열을 출력합니다. (결과: {"name":"Hwang","age":30})
    }
    }

++연산자 (JSON)


+ 연산자는 변수의 값을 1씩 증가시키는 단항 연산자입니다.
이 연산자는 변수를 변경하는 역할을 합니다. 변수에 할당된 값을 증가시키는 것이 주요 기능이지만, 약간의 차이가 있습니다. 

{
    산술 연산자(Arithmetic Operators): 수학적 계산을 수행합니다. 예를 들어, 덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/) 등이 있습니다. 
    할당 연산자(Assignment Operators): 변수에 값을 할당합니다. 예를 들어, =, +=, -=, *=, /= 등이 있습니다.
    비교 연산자(Comparison Operators): 두 값을 비교하여 참(true) 또는 거짓(false)을 반환합니다. 예를 들어, ==, !=, ===, !==, >, <, >=, <= 등이 있습니다.
    논리 연산자(Logical Operators): 논리적인 연산을 수행하여 참 또는 거짓을 반환합니다. 주로 조건문과 함께 사용됩니다. 예를 들어, &&(AND), ||(OR), !(NOT) 등이 있습니다.
    비트 연산자(Bitwise Operators): 이진수를 다루는 연산을 수행합니다. 예를 들어, &(AND), |(OR), ^(XOR), ~(NOT), <<(왼쪽 시프트), >>(오른쪽 시프트) 등이 있습니다.
    삼항 연산자(Ternary Operator): 조건문의 간단한 버전으로, 조건에 따라 두 개의 표현식 중 하나를 선택하여 실행합니다. 예를 들어, 조건 ? 값1 : 값2 형식으로 사용됩니다.
    타입 연산자(Type Operators): 피연산자의 데이터 타입을 확인하거나 변환하는데 사용됩니다. 예를 들어, typeof 연산자는 피연산자의 데이터 타입을 반환합니다.
}

++자료형 (JSON)


- 자료의 형태
- 총 6가지로 분류(string, number, boolean, function, object, undefined)
- 이 중 숫자, 문자열, 부울은 기본 자료형 


원시 자료형(Primitive Data Types):

숫자(Number): 정수와 부동소수점 숫자를 포함합니다. 예를 들어, 5, 3.14 등이 있습니다.
문자열(String): 텍스트 데이터를 나타냅니다. 예를 들어, "Hello, world!" 등이 있습니다.
불리언(Boolean): 참(true) 또는 거짓(false)을 나타냅니다. 예를 들어, true, false 등이 있습니다.
null: 값이 없음을 나타냅니다. 존재하지 않는 값이거나 비어 있는 값을 나타냅니다.
undefined: 값이 정의되지 않음을 나타냅니다. 변수가 선언되었지만 값이 할당되지 않은 경우입니다.
심볼(Symbol): 고유하고 변경 불가능한 값으로 사용됩니다.
객체(Object): 여러 값을 그룹화하고 관리하기 위해 사용됩니다. 객체는 키-값 쌍의 집합으로 구성되며, 키를 사용하여 값을 접근합니다. 예를 들어, { name: "John", age: 30 } 등이 있습니다.

배열(Array): 여러 값을 순차적으로 저장하는 자료구조입니다. 배열은 대괄호([])로 둘러싸인 요소들의 집합으로 구성되며, 각 요소는 쉼표로 구분됩니다. 예를 들어, [1, 2, 3, 4, 5] 등이 있습니다.

함수(Function): 일련의 작업을 수행하는 코드 블록입니다. 함수는 재사용 가능한 코드 조각을 정의하고 호출하여 실행할 수 있습니다.

기타 자료형:

Date: 날짜와 시간을 나타내는 객체입니다.
RegExp: 정규 표현식을 나타내는 객체입니다.
Map, Set: 특정 데이터 구조를 나타내는 객체입니다.
이러한 자료형들은 JavaScript에서 데이터를 다루고 처리하는데 사용되며, 각각의 특징과 용도에 따라 다양하게 활용됩니다.
