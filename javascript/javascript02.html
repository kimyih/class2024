01.변수 : 데이터 불러오기


변수는 프로그래밍에서 데이터를 저장하고 참조하는데 사용되며, 값은 언제든지 변경될 수 있습니다.
변수를 선언할 때는 보통 데이터의 타입과 함께 이름을 지정하고 값을 할당합니다.
변수는 프로그램에서 재사용되며, 데이터를 보다 효율적으로 다루고 조작하는 데 도움이 됩니다.
JavaScript에서는 변수를 선언할 때 let, const, var 등의 키워드를 사용합니다. 

{
    let x = 100, y = 200, z = "javascript";

    console.log(x, y, z);
}
결과 확인하기
100
200
javascript

02.상수 : 데이터 불러오기


상수는 값을 한 번 할당하면 변경할 수 없는 변수와 유사한 개념입니다.
JavaScript에서는 const 키워드를 사용하여 상수를 선언합니다.
한 번 정해진 값을 다시 변경할 수 없으므로 코드의 안정성과 가독성을 높이는 데 활용됩니다.
주로 프로그램에서 변하지 않는 값들을 상수로 정의하여 사용합니다. 

{               
    const x = 100, y = 200, z = "javascript";

    console.log(x, y, z);         
}
결과 확인하기
100
200
javascript

03.배열 : 데이터 불러오기


배열은 데이터의 집합이라고 생각할 수 있습니다. 이 집합 안에는 여러 가지 항목들이 들어 있으며, 각 항목은 순서에 따라 정렬되어 있습니다.
JavaScript에서는 대괄호 []로 배열을 만들고, 각 항목은 쉼표로 구분됩니다.
배열은 데이터를 구조화하고, 배열의 요소에는 인덱스를 통해 접근할 수 있습니다.
이를 통해 프로그램에서 데이터를 효율적으로 관리하고 다룰 수 있습니다. 

{
    const arr = [100, 200, "javascript"];

    console.log(arr[0], arr[1], arr[2]);
}
결과 확인하기
100
200
javascript

배열의 "갯수"는 배열에 포함된 요소의 총 개수를 나타냅니다.
JavaScript에서는 배열의 길이를 알아보기 위해 배열의 length 속성을 사용합니다.
이 속성을 통해 배열의 갯수를 쉽게 확인할 수 있습니다.
배열의 갯수를 알면 배열 안의 데이터를 효율적으로 관리하고 조작할 수 있습니다.

    console.log(arr.length);
}

05.배열 : 데이터 불러오기 : for()문


for()문은 반복 작업을 수행하는 제어 구조로, 초기값, 조건식, 증감식을 이용하여 반복의 시작과 종료를 결정하고, 각 반복마다 실행할 코드를 지정합니다.
초기값은 반복 시작을 정하고, 조건식은 반복을 계속할지 결정하며, 증감식은 반복마다 변수의 값을 변경합니다.
이를 통해 반복 작업을 효율적으로 처리할 수 있습니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
    console.log(arr[3]);
    console.log(arr[4]);
    console.log(arr[5]);
    console.log(arr[6]);
    console.log(arr[7]);
    console.log(arr[8]);

    for (let i = 0; i < arr.length; i++) {              // 초깃값; 조건식; 증감값;
        console.log(arr[i]);
    }
}

06.배열 : 데이터 불러오기 : forEach()


forEach() 메서드는 배열의 각 요소에 대해 순차적으로 작업을 수행할 때 사용되는 배열 메서드입니다.
이 메서드는 주어진 배열을 처음부터 끝까지 순회하며 각 요소에 대해 지정된 작업(콜백 함수)을 실행합니다.
콜백 함수는 배열의 요소 값과 인덱스를 인자로 받아 사용하며, 배열 자체도 인자로 전달됩니다.
forEach() 메서드를 사용하면 반복문을 대체하여 코드를 더 간결하게 작성할 수 있습니다.
이를 통해 배열을 순회하고 처리할 때 반복문을 사용하는 것보다 코드가 더욱 명확하고 읽기 쉬워집니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];
    // 01
    console.log("06. 배열 : 데이터 불러오기 : forEach()");
    arr.forEach(function (element) {    //순서대로 작동
        console.log(element);
    });
    // 02
    arr.forEach(function (element, index) {     //순서대로 작동
        console.log(index);
    });
    // 02
    arr.forEach(function (element, index, array) {  //순서대로 작동
        console.log(array);
    });
    //화살표 함수
    // 01
    arr.forEach((e) => {
        console.log(e);
    });
    // 02
    arr.forEach(e => {
        console.log(e);
    });
    // 03
    arr.forEach(e => console.log(e))
}
결과 확인하기
100~900
0~8

07.배열 : 데이터 불러오기 : for of


for...of 반복문은 배열의 각 항목을 하나씩 순서대로 가져와서 사용할 때 쓰는 도구입니다.
배열에 있는 요소들을 한 번에 하나씩 가져와서 사용할 수 있습니다.
이 반복문은 반복할 때마다 배열의 다음 요소를 가져오기 때문에 따로 인덱스를 신경쓰지 않아도 됩니다.
즉, 배열을 편하게 순회하고 요소를 활용할 수 있습니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    console.log("07. 배열 : 데이터 불러오기 : for of");
    for (let element of arr) {
        console.log(element);
    }
}
결과 확인하기
100 ~ 700
