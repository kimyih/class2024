01. 함수 : 선언적 함수


선언적 함수는 구현 세부사항을 숨기고 목적을 명확히 전달하는 함수 스타일입니다.
이러한 방식은 추상화된 인터페이스를 제공하여 코드의 가독성과 유지보수성을 향상시킵니다.
함수형 프로그래밍의 중요한 개념 중 하나로, 코드를 이해하기 쉽고 변경하기 용이하게 합니다. 

{
    function func() {
        console.log("01. 함수가 실행되었습니다.");
    }
    func();
}

02. 함수 : 익명 함수


익명 함수는 이름이 없는 함수로, 주로 변수에 할당되거나 다른 함수의 인자로 전달되어 사용됩니다.
일급 객체로 취급되어 변수에 할당하거나 콜백 함수로 활용되며, 스코프 내에서만 영향을 미칩니다.
함수 표현식을 사용하여 정의되며, 코드의 모듈화와 스코프 충돌을 방지하는 데 유용합니다. 

{
    const func = function () {
        console.log("02. 함수가 실행되었습니다.");
    }
    func();
}

03. 함수 : 매개변수 함수


매개변수 함수는 다른 함수의 매개변수로 전달되는 함수를 가리킵니다.
이는 주로 콜백 함수로 활용되며, 함수의 동작을 확장하거나 조절하는 데 사용됩니다.
매개변수 함수를 이용하면 코드의 재사용성과 유연성을 높일 수 있으며, 함수형 프로그래밍에서는 코드를 더욱 간결하고 추상화된 수준으로 작성할 수 있습니다. 

{
   // 선언적 함수
    function func(str) {
        console.log(str);
    }
    func("03. 함수가 실행되었습니다.");

    //익명 함수
    const func1 = function (str) {
        console.log(str);
    }
    func("03. 함수가 실행되었습니다.");
}

06. 화살표 함수 : 선언적 함수


화살표 함수는 ES6에서 도입된 간결한 함수 정의 방식으로, 일반 함수보다 구문이 간결하며 주로 익명 함수로 사용됩니다.
외부 스코프의 this를 그대로 사용하므로 this 바인딩에 대한 혼란을 줄일 수 있습니다.
한 줄의 표현식에서는 중괄호와 return 문을 생략하고 자동으로 반환됩니다. 

{

    }
    func();

    // 선언적 함수를 화살표 함수로 바꿈
    func = () => {
        console.log("06. 함수가 실행되었습니다.")
    }
    func();
}

주로 다른 함수의 매개변수로 전달되어 사용되며, 간결한 문법으로 함수를 정의하고 실행할 수 있습니다.
이를 통해 코드를 간결하게 작성하고 가독성을 높일 수 있습니다. 

{
    // 익명 함수(화살표 함수) + 매개변수 + 리턴값
    const func = (str) => {
        return str;
    }
    console.log(func("10. 함수가 실행되었습니다."));

    // 매개변수가 하나이면 괄호 생략 가능

    // 리턴 생략
    const func3 = str => str;
    console.log(func3("10. 함수가 실행되었습니다."));

    // 선언적 함수(가독성X)
    func4 = str => str;
    console.log(func4("10. 함수가 실행되었습니다."));
}

11. 함수 유형 : 함수와 매개변수를 이용한 형태


함수는 프로그래밍에서 재사용 가능한 코드 조각으로, 특정 작업을 수행하는 데 사용됩니다.
매개변수는 함수에 전달되는 값으로, 함수가 작업을 수행하는 데 필요한 정보를 제공합니다.
함수는 매개변수를 이용하여 입력값을 받아들이고, 이를 기반으로 작업을 수행한 후 결과값을 반환할 수 있습니다.
이를 통해 함수는 다양한 입력값을 받아들여 다양한 결과를 생성할 수 있습니다.

예를 들어, 덧셈을 수행하는 함수는 두 개의 숫자를 매개변수로 받아들이고, 이를 더한 후 그 결과를 반환할 수 있습니다.
이러한 방식으로 함수는 매개변수를 이용하여 작업을 유연하게 조절할 수 있습니다.
또한 매개변수는 함수 호출 시에 값을 전달하는데 사용되므로, 함수의 재사용성을 높이는 데 중요한 역할을 합니다.

함수와 매개변수를 이용하면 코드를 더 간결하고 모듈화할 수 있으며, 코드의 가독성과 유지보수성을 향상시킬 수 있습니다.
따라서 함수와 매개변수는 프로그래밍에서 핵심적인 개념으로, 프로그램을 구조화하고 효율적으로 관리하는 데 도움을 줍니다.
함수와 매개변수의 사용은 대부분의 프로그래밍 언어에서 지원되며, 이를 통해 다양한 프로그램을 개발할 수 있습니다. 

{
    // 함수 정의: func이라는 이름의 함수를 정의합니다. 이 함수는 num과 str이라는 두 개의 매개변수를 받습니다.
    function func(num, str) {
        // 첫 번째 출력: 숫자(num)와 문자열(str)을 결합하여 출력합니다.
        console.log(num + ". " + str);
        // 두 번째 출력: 숫자(num)와 문자열(str)을 템플릿 리터럴을 이용하여 출력합니다.
        console.log(`${num}. ${str}`);
    }
    // 함수 호출: func 함수를 호출합니다. 숫자 11과 문자열 "함수가 실행되었습니다."를 매개변수로 전달합니다
    func(11, "함수가 실행되었습니다.")             
}
결과 확인하기
11. 함수가 실행되었습니다.

12. 함수 유형 : 함수와 변수를 이용한 형태


함수와 변수를 이용한 형태는 프로그래밍에서 데이터를 다루고 특정 작업을 수행하는 핵심적인 방법입니다.
변수는 값이 저장되는 메모리 공간으로, 함수 내에서 사용될 수 있습니다.
함수는 특정 작업을 수행하는 코드 블록으로, 변수를 활용하여 작업을 수행하고 결과를 반환할 수 있습니다.
함수 내부에서 선언된 변수는 지역 변수로, 함수 내에서만 유효하며 함수가 실행될 때 생성되고 종료될 때 사라집니다.
반면에 전역 변수는 프로그램 전체에서 사용 가능하며 어디서든 접근할 수 있습니다.
함수와 변수를 함께 사용하면 코드를 모듈화하여 관리하기 쉽게 만들어줍니다.
이를 통해 코드의 가독성과 유지보수성을 향상시킬 수 있습니다.

{
    const num = 12;     // 상수 num에 12를 할당합니다.
    const str = "함수가 실행되었습니다.";   // 상수 str에 "함수가 실행되었습니다."를 할당합니다.

    function func(num, str) {       // 함수 정의: func이라는 이름의 함수를 정의합니다. 이 함수는 num과 str이라는 두 개의 매개변수를 받습니다.
        console.log(`${num}. ${str}`);       // console.log를 이용하여 템플릿 리터럴을 통해 num과 str을 결합하여 출력합니다.
    }
    // 함수 호출: func 함수를 호출합니다. num과 str을 인자로 전달하여 함수를 실행합니다.
    func(num, str);
}

18. 함수 유형 : 프로토타입 함수


프로토타입 함수는 JavaScript에서 객체 지향 프로그래밍을 구현하는 중요한 개념 중 하나입니다.
JavaScript는 프로토타입 기반의 언어로, 모든 객체는 프로토타입을 가지고 있습니다.
이 때, 프로토타입 객체에 메서드를 추가하면, 해당 메서드는 생성된 모든 객체들에게 공유됩니다.
이를 통해 코드의 중복을 줄이고 메모리를 효율적으로 사용할 수 있습니다.

프로토타입 함수를 사용하여 객체에 메서드를 추가할 때, 모든 객체가 이 메서드를 공유하게 됩니다.
따라서 객체가 생성될 때마다 해당 메서드가 재생성되는 것이 아니라, 프로토타입 객체에 메서드가 한 번만 생성되어 모든 객체가 공유하게 됩니다.
이는 객체 생성 시 메모리 사용을 최적화하고, 유지보수성을 향상시키는데 도움이 됩니다.

프로토타입 함수는 JavaScript에서 객체 지향 프로그래밍을 할 때 매우 유용하며, 코드의 재사용성을 높이고 효율적인 메모리 사용을 가능하게 합니다. 

{
    // 함수 정의
    function Func(num, str) {
        this.num = num;
        this.str = str;
    }

    // 메서드 추가
    Func.prototype.result = function () {
        console.log(`${this.num}. ${this.str}`);
    }

    // 인스턴스 생성
    const info1 = new Func(21, "함수가 실행되었습니다.");
    const info2 = new Func(22, "함수가 실행되었습니다.");

    // 호출
    info1.result();
    info2.result();
}
