01. .at()


"문자열".at(위치값)

  • 지정한 숫자(index)를 받아, 문자열에서 해당되는 인덱스의 요소를 반환합니다.
  • 대소문자를 구별합니다.
  • 문자열의 길이보다 큰 인덱스를 지정하면 undefined를 반환합니다.
  • 음수인 경우 문자열의 오른족에서부터 인덱스를 셉니다.

{
        "javascript reference".at()       //j
        "javascript reference".at(0)      //j
        "javascript reference".at(1)      //a
        "javascript reference".at(2)      //v
        "javascript reference".at(3)      //a
        "javascript reference".at(4)      //s
        "javascript reference".at(5)      //c
        "javascript reference".at(-1)     //e
        "javascript reference".at(-2)     //c
        "javascript reference".at(-3)     //n
        "javascript reference".at(-4)     //e
        "javascript reference".at(-5)     //r
        "javascript reference".at(100)    // undefined
}

02. .includes()


"문자열".includes(값)

  • 'String.prototype.includes()' 메서드는 문자열 안에 특정 문자열이나 문자가 있는지 확인하는 JavaScript의 기능입니다.
  • 이 메서드는 그저 '예'나 '아니오'의 답을 반환합니다.

{
        const str = 'Hello, world!';
        const includes1 = str.includes('o');
        console.log(includes1); // 출력 결과: true
        
        const includes2 = str.includes('x');
        console.log(includes2); // 출력 결과: false
}

위의 예제에서 첫 번째 줄에서 'Hello, world!'라는 문자열에 'o'라는 문자가 포함되어 있는지 확인하고 있습니다.
만약에 포함되어 있다면, includes1은 true가 됩니다.
두 번째 줄에서는 'x'라는 문자가 포함되어 있는지 확인하고 있습니다.
포함되어 있지 않기 때문에 includes2는 false가 됩니다.

03. .indexof()


"문자열".includes(값)

  • 문자열 검색: 이 메서드는 문자열 안에서 특정 문자열 또는 문자의 첫 번째 발생 위치를 찾습니다.
  • 인덱스 반환: 찾는 문자열이나 문자가 문자열 안에서 발견되면 해당 인덱스를 반환합니다. 발견되지 않을 경우에는 -1을 반환합니다.
  • 대소문자 구분: 기본적으로 'indexOf()'는 대소문자를 구분합니다. 예를 들어, 'Hello' 문자열에서 'h'를 찾으려고 할 때 대소문자를 구분하므로 'indexOf()'는 발견되지 않습니다.
  • 선택적 시작 인덱스: 선택적으로 시작 인덱스를 지정하여 검색을 시작할 위치를 설정할 수 있습니다. 이를 통해 특정 위치 이후에서부터 검색을 시작할 수 있습니다.
  • 첫 번째 발견 위치 반환: 만약 검색 문자열이나 문자가 문자열 안에 여러 번 나타날 경우에도 첫 번째 발견된 위치의 인덱스만 반환합니다.
  • 인덱스 반환: 찾는 문자열이나 문자가 문자열 안에서 발견되면 해당 인덱스를 반환합니다. 발견되지 않을 경우에는 -1을 반환합니다.
  • 원본 문자열 변경 없음: 'indexOf()' 메서드는 원본 문자열을 변경하지 않습니다. 따라서 메서드를 호출하더라도 문자열 자체는 변경되지 않습니다.

{
        const str = 'Hello, world!';
        const index1 = str.indexOf('o'); // 'o'의 첫 번째 위치를 찾습니다.
        console.log(index1); // 출력 결과: 4
        
        const index2 = str.indexOf('l'); // 'l'의 첫 번째 위치를 찾습니다.
        console.log(index2); // 출력 결과: 2
        
        const index3 = str.indexOf('x'); // 'x'를 찾지 못하므로 -1을 반환합니다.
        console.log(index3); // 출력 결과: -1                                        
}

'String.prototype.indexOf()' 메서드는 문자열에서 특정 문자열 또는 문자의 위치를 찾는 데 사용되며, 검색된 문자열이나 문자가 발견되지 않을 경우 -1을 반환합니다.

04. .match()


"문자열".match(정규식)

  • 'String.prototype.match()' 메서드는 정규 표현식을 사용하여 문자열에서 패턴을 검색하고, 검색된 결과를 배열로 반환하는 JavaScript의 문자열 메서드입니다. .
  • 이 메서드는 대상 문자열과 일치하는 패턴이 발견되면 모든 일치 항목을 배열로 반환하고, 일치하는 항목이 없으면 null을 반환합니다. 

{
        const str = 'The rain in Spain falls mainly on the plain';
        const matches = str.match(/ain/g);
        console.log(matches); // 출력 결과: ["ain", "ain", "ain", "ain"]
        
        // 위의 예제에서는 'ain'이라는 패턴과 일치하는 모든 부분 문자열을 검색하고 있습니다. 
        // 'g' 플래그는 전역 검색을 의미하며, 따라서 대상 문자열 내에서 모든 일치 항목을 찾습니다. 
        // 이렇게 찾은 모든 일치 항목은 배열로 반환되어 matches 변수에 할당됩니다.
}

'String.prototype.match()' 메서드는 문자열에서 특정 패턴을 찾아 그 결과를 배열로 반환하는 JavaScript의 기능입니다.
예를 들어, "Hello, world!"라는 문자열에서 "Hello"라는 부분을 찾고 싶다면, 'Hello, world!'.match(/Hello/);와 같이 사용하면 됩니다.
이렇게 하면 ["Hello"]이라는 배열이 반환 됩니다.
만약 찾는 패턴이 없으면 null이 반환이 됩니다.
요약하자면, 'String.prototype.match()' 메서드는 문자열에서 패턴을 찾아서 그 결과를 배열로 반환하는 겁니다.

05. .search()


"문자열".search(정규식)

  • 'String.prototype.search()' 메서드는 주어진 정규 표현식과 일치하는 문자열의 위치를 검색하는 JavaScript의 문자열 메서드입니다.
  • 이 메서드는 일치하는 문자열이 발견되면 해당하는 첫 번째 위치의 인덱스를 반환하고, 일치하는 문자열이 발견되지 않으면 -1을 반환합니다. 

{
        const str = 'Hello, world!';
        const position = str.search('world');
        console.log(position); // 출력 결과: 7
        
        // 위의 예제에서는 'Hello, world!' 문자열에서 'world'라는 문자열을 찾고 있습니다. 
        // 'world' 문자열은 인덱스 7부터 시작되므로, search() 메서드는 7을 반환합니다.

        const str = 'The rain in Spain falls mainly on the plain';
        const position = str.search(/ain/g);
        console.log(position); // 출력 결과: 5

        // 위의 예제에서는 'ain'이라는 패턴을 찾고 있습니다. 
        // 이 패턴은 'The rain in Spain falls mainly on the plain' 문자열에서 'ain'이 처음 나타나는 위치는 인덱스 5입니다. 
        // 따라서 search() 메서드는 5를 반환합니다.
}

search() 메서드는 문자열에서 특정 패턴을 검색할 때 사용되며, 발견된 패턴의 첫 번째 위치를 반환합니다.

06. .slice()


"배열 또는 문자열".slice(시작 인덱스, 종료 인덱스)

  • 'String.prototype.slice()' 메서드는 문자열에서 특정 부분을 추출하여 새로운 문자열을 반환하는 JavaScript의 문자열 메서드입니다.
  • 이 메서드는 원본 문자열을 변경하지 않고, 추출한 부분 문자열을 반환합니다.
  • slice() 메서드는 원본 문자열을 변경하지 않고, 특정 부분을 추출하여 새로운 문자열을 반환하기 때문에, 문자열을 수정하지 않고 특정 부분을 추출하는 데에 유용합니다.

{
        const str = 'Hello, world!';
        const slicedStr = str.slice(7, 12);
        console.log(slicedStr); // 출력 결과: 'world'                                        
}

위의 예제에서는 'Hello, world!' 문자열에서 인덱스 7부터 12 이전까지의 부분 문자열을 추출하고 있습니다. 따라서 slice() 메서드는 'world'라는 부분 문자열을 반환합니다. 

{
        const str = 'Hello, world!';
        const slicedStr1 = str.slice(-6); // 끝에서부터 6글자 추출
        console.log(slicedStr1); // 출력 결과: 'world!'
        
        const slicedStr2 = str.slice(7); // 인덱스 7부터 끝까지 추출
        console.log(slicedStr2); // 출력 결과: 'world!'                                                                               
}

이 메서드는 두 개의 매개변수를 사용합니다. 첫 번째 매개변수는 추출을 시작할 인덱스를 나타내며, 두 번째 매개변수는 추출을 끝낼 인덱스를 나타냅니다.
두 번째 매개변수는 선택적으로 사용할 수 있으며, 지정하지 않으면 문자열의 끝까지 추출됩니다.
음수 인덱스를 사용하여 문자열의 끝에서부터 추출할 수도 있습니다.

07. .substring()


"문자열".substring(시작 인덱스, 종료 인덱스)

  • substring() 메서드는 자바스크립트의 String 객체에서 사용되며, 지정된 시작 인덱스와 종료 인덱스 사이의 부분 문자열을 추출하여 새로운 문자열로 반환합니다.
  • 이 메서드는 원본 문자열을 변경하지 않습니다.
  • 메서드는 문자열을 처리할 때 유용하게 사용되며, 특정 부분의 문자열을 필요로 할 때 적합합니다.

{
        const newString = str.substring(indexStart, indexEnd)                                                                                                           
}

indexStart: 추출을 시작할 문자열의 인덱스입니다.
indexEnd: 추출을 종료할 문자열의 인덱스로, 이 인덱스의 문자는 포함되지 않습니다. indexEnd가 생략되면 문자열의 끝까지 추출됩니다. 

{
        const text = 'Hello, world!';

        // 인덱스 7부터 12까지의 부분 문자열을 추출합니다.
        const part = text.substring(7, 12);
        
        console.log(part);
        // 출력: 'world'                                                                                                                                                                                                 
}

이 예제에서는 text 문자열의 인덱스 7에서 12까지의 부분을 추출하여 'world'라는 새로운 문자열을 생성합니다. indexEnd에 해당하는 문자 '!'는 포함되지 않습니다.

결과 확인하기
'morning, everyone!'

08. .split


"문자열".indexof(옵션)

  • 'String.prototype.split()' 메서드는 문자열을 특정 구분자를 기준으로 여러 부분으로 나누어 배열로 만드는 JavaScript의 문자열 메서드입니다.
  • split() 메서드는 문자열을 구분자를 기준으로 나누어 배열로 반환하므로, 데이터를 처리하거나 분석할 때 유용하게 사용됩니다.
  • split() 메서드는 첫 번째 매개변수로 구분자를 받으며, 두 번째 매개변수로는 배열의 최대 길이를 지정할 수 있습니다. 만약 두 번째 매개변수를 생략하면 모든 부분 문자열이 포함된 배열이 반환됩니다.

{
        const str = 'apple,banana,orange';
        const fruits = str.split(',');
        console.log(fruits); // ["apple", "banana", "orange"]                                                                                                                                                    
}

위의 예제에서는 최대 길이를 2로 지정하여 쉼표를 기준으로 최대 2개의 부분 문자열로 나누고 있습니다.
따라서 split() 메서드는 ["apple", "banana"] 배열을 반환합니다.

{
        const str = 'apple,banana,orange';
        const fruits1 = str.split(',', 2);
        console.log(fruits1); // ["apple", "banana"]                                                                                                                 
}

위의 예제에서는 최대 길이를 2로 지정하여 쉼표를 기준으로 최대 2개의 부분 문자열로 나누고 있습니다.
따라서 split() 메서드는 ["apple", "banana"] 배열을 반환합니다. 

{
        const str = 'apple,banana,orange';
        const fruits2 = str.split(/[, ]+/);
        console.log(fruits2); // ["apple", "banana", "orange"]                                                                                                                                                      
}

위의 예제에서는 쉼표(',') 또는 공백(' ')을 구분자로 사용하여 문자열을 나누고 있습니다.
따라서 split() 메서드는 ["apple","banana", "orange"] 배열을 반환합니다.

결과 확인하기
["apple", "banana", "orange"]
["apple", "banana"]

09. .trim


"문자열".trim()

  • 'String.prototype.trim()' 메서드는 문자열의 양 끝에서 공백을 제거한 새로운 문자열을 반환하는 JavaScript의 문자열 메서드입니다.
  • trim() 메서드는 문자열의 양 끝에서 공백(스페이스, 탭, 개행)을 제거합니다. 중간에 있는 공백은 제거하지 않습니다.
  • trim() 메서드는 주로 사용자 입력이나 데이터 처리에서 문자열의 앞뒤 공백을 제거하여 데이터의 일관성을 유지하고 문자열을 다루기 쉽게 만들 때 유용하게 사용됩니다.

{
        const str = '   Hello, world!   ';
        const trimmedStr = str.trim();
        console.log(trimmedStr); // "Hello, world!"                                                                                                                                                                                                                          
}

위의 예제에서는 문자열 ' Hello, world! '에서 양 끝의 공백을 제거하고 있습니다.
따라서 trim() 메서드는 'Hello, world!'라는 새로운 문자열을 반환합니다.

{
        const str = '   Hello,      world!   ';
        const trimmedStr = str.trim();
        console.log(trimmedStr); // "Hello,      world!"                                                                                                                                                       
}

위의 예제에서는 문자열 ' Hello, world! '에서 양 끝의 공백을 제거하고 있습니다. 중간에 있는 공백은 제거되지 않고 유지됩니다.

결과 확인하기
"Hello, world!"
"Hello, world!"
