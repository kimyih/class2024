01. .find()


.find()

  • 'Array.prototype.find()' 메서드는 주어진 조건을 만족하는 배열의 첫 번째 요소를 찾아 반환하는 JavaScript의 배열 메서드입니다.
  • find() 메서드는 배열에서 주어진 조건을 만족하는 첫 번째 요소를 찾아 반환하는데 사용됩니다.
    만약 조건을 만족하는 요소가 없을 경우 undefined를 반환합니다.

{
    const numbers = [1, 2, 3, 4, 5];

    const found = numbers.find(function(element) {
        return element > 3;
    });
    
    console.log(found); // 4                                                                                                                                                                                                   
}

위의 예제에서는 [1, 2, 3, 4, 5]라는 배열에서 조건을 만족하는 첫 번째 요소를 찾고 있습니다. 조건은 각 요소가 3보다 큰지를 확인하는 것입니다.
따라서 find() 메서드는 4라는 값을 반환합니다

find() 메서드는 주어진 조건 함수를 사용하여 배열의 각 요소를 순회하며, 조건 함수가 true를 반환하는 첫 번째 요소를 찾습니다.
찾은 요소를 반환하고, 만약 조건을 만족하는 요소가 없을 경우 undefined를 반환합니다.

{
    const people = [
    { name: 'Alice', age: 30 },
    { name: 'Bob', age: 25 },
    { name: 'Charlie', age: 35 }
    ];
    
    const foundPerson = people.find(function(person) {
    return person.age === 25;
    });
    
    console.log(foundPerson); // { name: 'Bob', age: 25 }                                                                                                                                                                    
}

위의 예제에서는 객체로 이루어진 배열에서 age가 25인 첫 번째 사람을 찾고 있습니다.
따라서 find() 메서드는 { name: 'Bob', age: 25 }라는 객체를 반환합니다.

02. .forEach()


.forEach()

  • 'Array.prototype.find()' 메서드는 주어진 조건을 만족하는 배열의 첫 번째 요소를 찾아 반환하는 JavaScript의 배열 메서드입니다.
  • forEach() 메서드는 주로 배열의 모든 요소에 대해 특정 작업을 수행할 때 사용됩니다. 예를 들어, 배열의 요소를 화면에 출력하거나, 배열의 요소를 가지고 연산을 수행하는 등의 작업에 활용될 수 있습니다.

{
    const numbers = [1, 2, 3, 4, 5];

    numbers.forEach(function(element) {
        console.log(element);
    });                                                                                                                                                                                                                                 
}

위의 예제에서는 [1, 2, 3, 4, 5]라는 배열의 각 요소에 대해 주어진 함수를 실행하고 있습니다. 각 요소는 순차적으로 함수의 매개변수로 전달되어 console.log()를 통해 출력됩니다.
forEach() 메서드는 배열을 순회하면서 각 요소에 대해 주어진 함수를 실행합니다. 이 때, 배열의 원본은 변경되지 않습니다. 

{
    const fruits = ['apple', 'banana', 'cherry'];

    fruits.forEach(function(fruit, index, array) {
        console.log(`Index ${index}: ${fruit}`);
    });
    
    // 출력 결과:
    // Index 0: apple
    // Index 1: banana
    // Index 2: cherry                                                                                                                                                                                      
}

1. currentValue: 배열에서 현재 순회 중인 요소의 값
2. index (선택적): 현재 순회 중인 요소의 인덱스
3. array (선택적): 원본 배열

03. .map()


.map()

  • 'Array.prototype.map()' 메서드는 배열의 각 요소에 대해 주어진 함수를 호출하여 새로운 배열을 생성하는 JavaScript의 배열 메서드입니다.
  • map() 메서드는 원본 배열을 변경하지 않고, 각 요소에 대해 주어진 함수를 호출하여 새로운 배열을 반환합니다.
  • map() 메서드는 배열의 모든 요소에 대해 어떤 작업을 수행하고 그 결과를 새로운 배열로 만들 때 사용됩니다.
    예를 들어, 배열의 요소를 변형하거나 각 요소에 대해 특정 연산을 수행하여 새로운 배열을 만들 때 유용하게 사용됩니다.

{
    const numbers = [1, 2, 3, 4, 5];

    const doubledNumbers = numbers.map(function(element) {
        return element * 2;
    });
    
    console.log(doubledNumbers); // [2, 4, 6, 8, 10]                                                                                                                                                                                                                                        
}

위의 예제에서는 [1, 2, 3, 4, 5]라는 배열의 각 요소에 대해 주어진 함수를 호출하여 새로운 배열을 생성하고 있습니다.
각 요소는 순차적으로 함수의 매개변수로 전달되며, 해당 요소를 2배로 만든 값으로 새로운 배열을 만들었습니다.

{
    const fruits = ['apple', 'banana', 'cherry'];

    const capitalizedFruits = fruits.map(function(fruit, index, array) {
        return fruit.toUpperCase();
    });
    
    console.log(capitalizedFruits); // ["APPLE", "BANANA", "CHERRY"]                                                                                                                                                                                                    
}

위의 예제에서는 각 과일의 첫 글자를 대문자로 만든 새로운 배열을 생성하고 있습니다.
1. currentValue: 배열에서 현재 순회 중인 요소의 값
2. index (선택적): 현재 순회 중인 요소의 인덱스
3. array (선택적): 원본 배열

결과 확인하기

["APPLE", "BANANA", "CHERRY"]

04. .indexOf()


.includes("문자열")

  • 'Array.prototype.indexOf()' 메서드는 배열에서 지정된 요소를 찾아 그 인덱스를 반환하는 JavaScript의 배열 메서드입니다.
  • indexOf() 메서드는 배열에서 요소를 검색하여 인덱스를 반환하는데 사용되며, 배열에 해당 요소가 있는지 여부를 확인하거나 특정 요소의 위치를 파악할 때 유용하게 사용됩니다.

{
    const fruits = ['apple', 'banana', 'cherry'];

    const index = fruits.indexOf('banana');
    console.log(index); // 1                                                                                                                                                                                                                                                            
}

위의 예제에서는 fruits 배열에서 'banana'라는 요소를 찾고 있습니다.
'banana'의 인덱스는 1이므로, indexOf() 메서드는 1을 반환합니다. 

{
    const fruits = ['apple', 'banana', 'cherry'];

    const index = fruits.indexOf('grape');
    console.log(index); // -1                                                                                                                                                                                                                       
}

indexOf() 메서드는 찾으려는 요소가 배열에서 첫 번째로 발견되는 위치의 인덱스를 반환합니다.
만약 배열 안에 해당 요소가 없으면 -1을 반환합니다.
위의 예제에서는 fruits 배열에서 'grape'라는 요소를 찾고 있습니다.
하지만 'grape'는 배열에 없으므로 indexOf() 메서드는 -1을 반환합니다. 

{
    const fruits = ['apple', 'banana', 'cherry', 'banana'];

    const index = fruits.indexOf('banana', 2);
    console.log(index); // 3                                                                                                                                                                                                                                    
}

indexOf() 메서드는 두 번째 매개변수로 검색을 시작할 인덱스를 지정할 수 있습니다. 이렇게 하면 해당 인덱스부터 배열의 끝까지만을 검색하게 됩니다.
위의 예제에서는 fruits 배열에서 인덱스 2부터 시작하여 'banana'라는 요소를 찾고 있습니다.
이 때, 두 번째 'banana'의 인덱스는 3이므로 indexOf() 메서드는 3을 반환합니다.

-1

05. .join()


.join()

  • Array.prototype.join() 메소드는 JavaScript에서 배열의 모든 요소를 문자열로 변환하고, 이를 연결하여 하나의 문자열로 합치는 기능을 합니다.
  • 이때, 각 요소 사이에 넣을 수 있는 구분자를 지정할 수 있습니다. 만약 구분자를 지정하지 않으면, 기본값으로 쉼표(,)가 사용됩니다.
  • join() 메서드는 배열의 모든 요소를 하나의 문자열로 결합할 때 사용됩니다. 주로 배열을 문자열로 변환하여 출력할 때나, CSV 파일을 생성할 때 유용하게 사용됩니다. 

{
    const fruits = ['apple', 'banana', 'cherry'];

    const joinedString = fruits.join(', ');
    console.log(joinedString); // "apple, banana, cherry"                                                                                                                                                                                                                                                              
}

위의 예제에서는 fruits 배열의 모든 요소를 쉼표와 공백으로 구분하여 하나의 문자열로 결합하고 있습니다.
따라서 join() 메서드는 "apple, banana, cherry"라는 문자열을 반환합니다. 

{
    const numbers = [1, 2, 3, 4, 5];

    const joinedString = numbers.join();
    console.log(joinedString); // "1,2,3,4,5"
}

join() 메서드는 배열의 각 요소를 문자열로 변환한 후, 지정된 구분자(separator)로 연결하여 하나의 문자열로 만듭니다.
구분자를 생략하면 기본값으로 쉼표(',')가 사용됩니다.
위의 예제에서는 numbers 배열의 모든 요소를 기본 구분자인 쉼표(',')로 연결하여 하나의 문자열로 만들고 있습니다.

결과 확인하기
"apple, banana, cherry"
"1,2,3,4,5"

06. .push()


.indexOf("문자열")

  • 'Array.prototype.push()' 메서드는 배열의 끝에 하나 이상의 요소를 추가하는 JavaScript의 배열 메서드입니다.
  • push() 메서드는 배열의 끝에 요소를 추가할 때 사용되며, 한 번에 여러 요소를 추가할 수도 있습니다.
    이 메서드를 사용하여 배열에 요소를 추가하면 배열의 원본이 변경됩니다.

{
    const fruits = ['apple', 'banana'];

    fruits.push('cherry');
    console.log(fruits); // ["apple", "banana", "cherry"]
}

위의 예제에서는 fruits 배열의 끝에 'cherry'라는 요소를 추가하고 있습니다.
따라서 push() 메서드를 호출한 후에는 fruits 배열이 ["apple", "banana", "cherry"]로 업데이트됩니다. 

{
    const numbers = [1, 2, 3];

    const newLength = numbers.push(4, 5);
    console.log(numbers); // [1, 2, 3, 4, 5]
    console.log(newLength); // 5                    
}

push() 메서드는 배열에 하나 이상의 요소를 추가하고, 배열의 길이를 반환합니다. 추가된 요소의 인덱스는 배열의 길이와 동일합니다.
위의 예제에서는 numbers 배열에 4와 5라는 요소를 추가하고 있습니다.
push() 메서드는 배열의 길이를 반환하므로, newLength 변수에는 배열의 새로운 길이인 5가 저장됩니다.

}

위의 예제에서는 fruits 배열의 끝에 'cherry'라는 요소를 추가하고 있습니다.
따라서 push() 메서드를 호출한 후에는 fruits 배열이 ["apple", "banana", "cherry"]로 업데이트됩니다. 

{
    const numbers = [1, 2, 3];

    const newLength = numbers.push(4, 5);
    console.log(numbers); // [1, 2, 3, 4, 5]
    console.log(newLength); // 5                    
}

push() 메서드는 배열에 하나 이상의 요소를 추가하고, 배열의 길이를 반환합니다. 추가된 요소의 인덱스는 배열의 길이와 동일합니다.
위의 예제에서는 numbers 배열에 4와 5라는 요소를 추가하고 있습니다.
push() 메서드는 배열의 길이를 반환하므로, newLength 변수에는 배열의 새로운 길이인 5가 저장됩니다.
