01.find


  • find() 메서드는 자바스크립트의 배열에서 사용할 수 있는 메서드로, 특정 조건(판별 함수)을 만족하는 배열 내의 첫 번째 요소를 찾아 그 값을 반환합니다.
  • 메서드는 배열을 순회하면서 각 요소에 대해 판별 함수를 실행하고, 그 결과가 true인 첫 번째 요소의 값을 반환합니다. 만약 어떤 요소도 조건을 만족하지 않으면 undefined를 반환합니다.

{
    const numbers = [4, 9, 16, 25, 29];

    // 판별 함수: 숫자가 10보다 큰 첫 번째 요소를 찾습니다.
    const firstLargeNumber = numbers.find(element => element > 10);
    
    console.log(firstLargeNumber);
    // 출력: 16                    
}

이 코드에서 numbers 배열의 각 요소는 차례로 판별 함수 element => element > 10에 의해 검사됩니다. 4, 9는 10보다 크지 않기 때문에 건너뛰고, 16은 10보다 크기 때문에 이 값을 반환하고 함수가 종료됩니다.

판별 함수
판별 함수는 각 배열 요소에 대해 호출되며, 다음 세 가지 매개변수를 받을 수 있습니다

element: 배열에서 현재 처리하고 있는 요소.
index: 처리하고 있는 요소의 인덱스.
array: 메서드가 호출된 배열.

02.forEach


  • forEach() 메서드는 자바스크립트의 배열에서 사용할 수 있는 메서드로, 배열의 각 요소에 대해 주어진 함수를 실행하는 데 사용됩니다.
  • forEach()는 결과를 반환하지 않습니다(즉, 반환 값은 undefined입니다).
    대신, 주어진 함수를 배열의 각 요소에 대해 한 번씩 실행하여 부작용(side effects)을 일으키는데 사용됩니다.

{
    // 사용 방법
    array.forEach(function(currentValue, index, arr), thisValue)          
}

function(currentValue, index, arr)는 각 요소에 대해 실행할 함수입니다. 이 함수는 다음 세 가지 인자를 받을 수 있습니다


currentValue: 배열에서 현재 처리 중인 요소의 값.
index: 현재 처리 중인 요소의 인덱스.
arr: 메서드가 호출된 배열.
thisValue는 (선택적으로) function에서 this로 사용될 값입니다. 

{
    const fruits = ["apple", "banana", "cherry"];

    fruits.forEach(function(item, index) {
        console.log(index, item);
    });
    
    // 출력:
    // 0 apple
    // 1 banana
    // 2 cherry                            
                }

이 코드에서 forEach()는 fruits 배열의 각 요소를 순서대로 접근하고, 각 요소의 이름과 인덱스를 콘솔에 출력합니다.

결과 확인하기

1 banana

03.filter


  • filter() 메서드는 자바스크립트의 배열에서 사용되며, 주어진 판별 함수를 만족하는 모든 요소를 새로운 배열로 반환합니다.
  • 이 메서드는 원본 배열을 변경하지 않고, 조건에 맞는 요소들만을 모아 새 배열을 만드는 데 사용됩니다

{
    // 사용 방법
    const newArray = array.filter(function(currentValue, index, arr), thisValue)     
}

function(currentValue, index, arr)는 각 요소에 대해 실행할 판별 함수입니다. 이 함수는 다음 세 가지 인자를 받을 수 있습니다

currentValue: 배열에서 현재 처리 중인 요소의 값.
index: 현재 처리 중인 요소의 인덱스.
arr: 메서드가 호출된 배열.
thisValue는 (선택적으로) function에서 this로 사용될 값입니다.
판별 함수는 각 요소에 대해 true 또는 false를 반환해야 하며, true를 반환하는 요소만 새로운 배열에 포함됩니다. 

{
    const numbers = [1, 2, 3, 4, 5, 6];

    // 짝수만 필터링해서 새 배열 생성
    const evenNumbers = numbers.filter(number => number % 2 === 0);
    
    console.log(evenNumbers);
    // 출력: [2, 4, 6]                                         
                }

이 코드에서 filter() 메서드는 numbers 배열의 각 요소가 짝수인지 확인하고, 짝수인 요소들만으로 구성된 새 배열 evenNumbers를 반환합니다.

결과 확인하기
[2,4,6]

04.map


  • map() 메서드는 자바스크립트에서 배열을 다룰 때 매우 유용하게 사용되는 메서드입니다.
  • 이 메서드는 배열의 모든 요소에 대해 주어진 함수를 호출하고, 그 결과를 모아 새로운 배열을 생성합니다
  • map() 메서드는 원본 배열을 변경하지 않으며, 각 요소에 대한 함수의 반환 값으로 구성된 새로운 배열을 반환합니다.

{
    // 사용 방법
    const newArray = array.map(function(currentValue, index, arr), thisValue)   
}

function(currentValue, index, arr)는 각 요소에 대해 실행할 함수입니다. 이 함수는 다음 세 가지 인자를 받을 수 있습니다

currentValue: 배열에서 현재 처리 중인 요소의 값.
index: 현재 처리 중인 요소의 인덱스.
arr: 메서드가 호출된 배열.
thisValue는 (선택적으로) function에서 this로 사용될 값입니다.
함수는 각 요소를 기반으로 새로운 값을 반환하며, 이 반환 값들은 새로운 배열에 순서대로 저장됩니다. 

{
    const numbers = [1, 2, 3, 4, 5];

    // 각 숫자의 제곱을 계산합니다.
    const squares = numbers.map(number => number * number);
    
    console.log(squares);
    // 출력: [1, 4, 9, 16, 25]                                                    
}

이 예제에서 map() 메서드는 numbers 배열의 각 요소를 제곱하여 squares라는 새 배열을 생성합니다.

결과 확인하기
[1, 4, 9, 16, 25]

05.includes


  • includes() 메서드는 자바스크립트에서 배열이 특정 요소를 포함하고 있는지 여부를 판별하는데 사용됩니다.
  • 이 메서드는 검색하고자 하는 요소가 배열 내에 존재하는 경우 true를 반환하고, 그렇지 않은 경우 false를 반환합니다.

{
    // 사용 방법
    const result = array.includes(valueToFind, fromIndex) 
}

valueToFind는 배열에서 찾고자 하는 값입니다.
fromIndex는 (선택적으로) 검색을 시작할 배열의 인덱스입니다.
이 값이 제공되지 않는 경우, 기본값은 0이며, 배열의 처음부터 검색이 시작됩니다.
음수 값이 주어진 경우, 배열의 끝에서부터 그 절대값만큼 떨어진 위치에서 검색이 시작됩니다. 

{
    const fruits = ['apple', 'banana', 'mango', 'orange'];

    // 'banana'가 배열에 포함되어 있는지 검사합니다.
    const hasBanana = fruits.includes('banana');
    
    console.log(hasBanana);
    // 출력: true                                                                    
}

이 예제에서 fruits 배열에 'banana'가 포함되어 있는지를 확인합니다. 'banana'가 존재하기 때문에 true가 출력됩니다.

결과 확인하기

06.indexOf


  • indexOf() 메서드는 자바스크립트에서 배열 내에서 특정 요소의 첫 번째 등장 위치를 찾아 그 인덱스를 반환합니다.
  • 이 메서드는 주어진 요소가 배열 내에 존재할 경우 해당 요소의 인덱스를 반환하고, 만약 배열 내에 해당 요소가 없다면 -1을 반환합니다.

{
    // 사용 방법
    const index = array.indexOf(searchElement, fromIndex)
}

searchElement는 배열에서 찾고자 하는 요소입니다.
fromIndex는 (선택적으로) 검색을 시작할 인덱스입니다.
이 값이 양수인 경우, 해당 인덱스부터 검색을 시작하며, 음수인 경우 배열의 끝에서부터 그 절대값만큼 떨어진 위치에서 검색이 시작됩니다.
fromIndex가 제공되지 않는 경우, 검색은 배열의 처음부터 시작됩니다. 

{
    const fruits = ['apple', 'banana', 'mango', 'orange'];

    // 'banana'의 위치를 찾습니다.
    const index = fruits.indexOf('banana');
    
    console.log(index);
    // 출력: 1                                                                                  
}

이 예제에서 fruits 배열 내에서 'banana'가 처음 등장하는 위치의 인덱스를 반환합니다.
'banana'는 배열의 두 번째 요소이므로, 인덱스 1이 출력됩니다.

07.join


  • join() 메서드는 자바스크립트에서 배열의 모든 요소를 연결하여 하나의 문자열로 만드는 데 사용됩니다.
  • 이 메서드는 주어진 구분자를 사용하여 배열의 각 요소 사이에 삽입하며, 결과적으로 단일 문자열을 반환합니다. 만약 구분자가 지정되지 않으면 기본적으로 쉼표(,)가 사용됩니다. 

{
    // 사용 방법
    const string = array.join(separator)
}

separator는 (선택적으로) 배열의 각 요소 사이에 삽입할 문자 또는 문자열입니다.
이 값이 제공되지 않는 경우, 요소들은 쉼표로 구분되어 연결됩니다.

{
    const elements = ['Fire', 'Air', 'Water', 'Earth'];

    // 기본 구분자(쉼표)로 요소들을 연결합니다.
    const result = elements.join();
    
    console.log(result);
    // 출력: 'Fire,Air,Water,Earth'                    
}

이 예제에서는 join() 메서드의 기본 구분자인 쉼표를 사용하여 elements 배열의 요소들을 연결합니다.
결과적으로 모든 요소가 쉼표로 구분된 단일 문자열이 반환됩니다.

08.push


  • push() 메서드는 자바스크립트에서 배열의 끝에 하나 이상의 요소를 추가하는 데 사용됩니다.
  • 이 메서드는 추가된 후의 배열의 새로운 길이를 반환하며, 원본 배열을 직접 변경합니다.

{
    // 사용 방법
    const newLength = array.push(element1, element2, ..., elementN)
}

element1, element2, ..., elementN은 배열의 끝에 추가할 요소들입니다.
메서드는 배열에 요소를 추가한 후, 변경된 배열의 새로운 길이를 반환합니다. 

{
    const numbers = [1, 2, 3];

    // 배열에 숫자 4를 추가합니다.
    const newLength = numbers.push(4);
    
    console.log(numbers);
    // 출력: [1, 2, 3, 4]
    console.log(newLength);
    // 출력: 4                                     
}

이 예제에서는 numbers 배열의 끝에 숫자 4를 추가합니다. push() 메서드는 요소가 추가된 후의 배열 길이인 4를 반환합니다.

결과 확인하기

09.reduce


  • reduce() 메서드는 자바스크립트에서 배열의 요소들을 하나의 값으로 줄여나가는 작업을 수행할 때 사용됩니다.
  • 이 메서드는 배열의 각 요소에 대해 주어진 리듀서(reducer) 함수를 실행하고, 그 결과를 누적하여 하나의 결과값을 반환합니다. reduce()는 배열 내 요소들을 합치거나, 요약하는 등의 연산에서 매우 유용하게 사용됩니다.

{
    // 사용 방법
    const result = array.reduce(function(accumulator, currentValue, currentIndex, array), initialValue)
}

function(accumulator, currentValue, currentIndex, array)는 배열의 각 요소에 대해 실행할 리듀서 함수입니다.

accumulator: 누적 계산의 결과값이며, 각 단계에서 리턴된 값이 다음 단계의 accumulator로 전달됩니다.
currentValue: 배열에서 현재 처리 중인 요소.
currentIndex (선택적): 현재 처리 중인 요소의 인덱스.
array (선택적): reduce() 메서드가 호출된 배열.
initialValue (선택적): 누적값의 초기값입니다. 이 값이 제공되면 첫 번째 호출에서 accumulator는 initialValue가 되고, 제공되지 않으면 배열의 첫 번째 요소가 초기값이 됩니다. 

{
    const numbers = [1, 2, 3, 4, 5];

    // 숫자들의 합을 계산합니다.
    const sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0);

    console.log(sum);
    // 출력: 15
}

이 예제에서는 numbers 배열의 모든 요소를 더하여 총합을 계산합니다. 초기값 0이 accumulator의 시작 값으로 설정되고, 배열의 각 요소가 차례로 더해져 최종적으로 15가 반환됩니다.

결과 확인하기
15

10.reverse


  • reverse() 메서드는 자바스크립트에서 배열의 요소들의 순서를 반대로 뒤집는 작업을 수행합니다.
  • 이 메서드는 원본 배열 자체를 수정하여 순서가 반대로 된 배열을 반환하며, 새로운 배열을 생성하지 않습니다.

{
    // 사용 방법
    const reversedArray = array.reverse()
}

이 메서드는 매개변수를 받지 않으며, 호출된 배열의 요소 순서를 뒤집고 그 배열을 그대로 반환합니다.

{
    const numbers = [1, 2, 3, 4, 5];

    // 배열의 요소 순서를 뒤집습니다.
    const reversedNumbers = numbers.reverse();
    
    console.log(reversedNumbers);
    // 출력: [5, 4, 3, 2, 1]
    console.log(numbers);
    // 출력: [5, 4, 3, 2, 1]                    
}

이 예제에서 numbers 배열의 요소 순서가 reverse() 메서드에 의해 반대로 뒤집혔습니다.
중요한 점은 reverse()가 원본 numbers 배열을 직접 변경한다는 것입니다.
결과적으로 뒤집힌 배열은 새로운 변수에 할당되었지만, 원본 배열 또한 변경되었습니다..

결과 확인하기
[5, 4, 3, 2, 1]
[5, 4, 3, 2, 1]

11.slice


  • slice() 메서드는 자바스크립트에서 배열의 특정 부분을 새로운 배열로 복사하여 반환하는 데 사용됩니다.
  • 이 메서드는 원본 배열을 변경하지 않고, 지정된 시작 인덱스와 종료 인덱스 사이의 요소들을 포함하는 새로운 배열을 생성합니다.

{
    // 사용 방법
    const newArray = array.slice(start, end)
}

start: 새로운 배열에 포함될 첫 번째 요소의 인덱스입니다.
이 인덱스는 포함되며, start가 생략되거나 undefined일 경우 0부터 시작합니다.
end: 새로운 배열에서 마지막으로 포함될 요소의 인덱스로, 이 인덱스의 요소는 포함되지 않습니다.
end가 생략되거나 undefined일 경우 배열의 끝까지 요소가 포함됩니다.

{
    const numbers = [1, 2, 3, 4, 5];

    // 인덱스 1부터 3까지의 요소를 추출합니다.
    const partOfNumbers = numbers.slice(1, 4);
    
    console.log(partOfNumbers);
    // 출력: [2, 3, 4]                                     
}

이 예제에서 slice() 메서드는 numbers 배열의 인덱스 1부터 3 (인덱스 4는 포함되지 않음)까지의 요소를 새로운 배열로 반환합니다.

결과 확인하기
[2, 3, 4]

12.splice


  • splice() 메서드는 자바스크립트에서 배열의 요소를 추가, 제거 또는 교체할 때 사용되며, 배열을 직접 변경합니다.
  • 이 메서드는 원하는 위치에서 요소를 추가하거나 제거할 수 있으며, 제거된 요소들을 새로운 배열로 반환합니다.

{
    // 사용 방법
    const removedElements = array.splice(start, deleteCount, item1, item2, ..., itemN)
}

start: 변경을 시작할 인덱스입니다.
음수인 경우 배열의 끝에서부터의 위치를 나타냅니다.
deleteCount: 제거할 요소의 수입니다. 이 값을 0으로 설정하면 요소가 제거되지 않고, 새 요소만 배열에 추가됩니다.
item1, item2, ..., itemN: 배열에 추가될 요소들입니다. 이 매개변수들은 선택적이며, 제공되지 않으면 요소들이 단순히 제거만 됩니다.

{
    const numbers = [1, 2, 3, 4, 5];

    // 인덱스 1에서 시작하여 2개의 요소를 제거합니다.
    const removed = numbers.splice(1, 2);
    
    console.log(numbers);
    // 출력: [1, 4, 5]
    console.log(removed);
    // 출력: [2, 3]                                                     
}

이 예제에서 splice() 메서드는 numbers 배열에서 인덱스 1부터 2개의 요소 (2와 3)를 제거합니다.
이 메서드는 제거된 요소들을 배열로 반환하며, 원본 배열은 변경됩니다.

결과 확인하기
[1, 4, 5]
[2, 3]

13.sort


  • sort() 메서드는 자바스크립트에서 배열의 요소들을 정렬하는 데 사용됩니다. 이 메서드는 원본 배열을 직접 수정하여 정렬된 배열을 반환합니다.
  • 본적으로 sort() 메서드는 요소들을 문자열로 변환하고 유니코드 코드 포인트의 순서대로 정렬합니다.
  • 이는 때때로 숫자 배열에서 예상치 못한 결과를 초래할 수 있기 때문에, 숫자를 정렬할 때는 정렬 순서를 정의하는 비교 함수를 제공해야 할 수 있습니다.

{
    // 사용 방법
    const sortedArray = array.sort(compareFunction)
}

compareFunction (선택적): 정렬 순서를 정의하는 함수입니다. 이 함수는 두 요소를 비교하고, 비교 결과에 따라 정렬 순서를 결정합니다.

비교 함수는 두 인자를 받아, 반환 값에 따라 요소의 순서를 결정합니다
반환 값이 0 미만이면 첫 번째 인자가 두 번째 인자보다 앞에 옵니다.
반환 값이 0이면 두 인자의 순서가 변경되지 않습니다.
반환 값이 0 초과이면 첫 번째 인자가 두 번째 인자보다 뒤에 옵니다. 

{
    const fruits = ['banana', 'apple', 'cherry'];

    // 문자열 정렬
    fruits.sort();
    
    console.log(fruits);
    // 출력: ['apple', 'banana', 'cherry']                                                                   
}

이 예제에서는 fruits 배열의 문자열 요소들을 알파벳 순서로 정렬합니다.

결과 확인하기
['apple', 'banana', 'cherry']
